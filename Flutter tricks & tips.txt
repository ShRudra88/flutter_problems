3. dependency vs dev_dependency
The fundamental distinction lies in whether the package is required for the application to function correctly in production. If it is, it's a dependency.
If it's only needed for development, testing, or build processes, it's a dev_dependency.

6.Most easy way to add firebase in flutter app
Step 1: Install Firebase CLI (Optional if using FlutterFire CLI only)
Step 2: Install FlutterFire CLI
Step 3: Create a Firebase Project
Step 4: Add Your App to Firebase
Step 5: Configure Firebase Automatically with CLI
Step 6: Add Firebase Dependencies
Step 7: Initialize Firebase in main.dart
Step 8: Run the App 
If it runs successfully → you have Firebase connected
Firebase makes Flutter app development easier by providing ready backend services like authentication, database, and cloud storage — 
so you don’t need to build your own server.
It helps you develop, deploy, and scale apps faster with real-time data and seamless cross-platform integration.
